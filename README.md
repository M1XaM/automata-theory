# Automata Theory Laboratory Works

This repository contains the laboratory works for the **Formal Languages & Finite Automata** course.

## [Lab 1: Intro to formal languages, Regular grammars, Finite Automata](lab1/)
**Description:** Introduction to formal languages. Conversion of a grammar to a finite automaton and string validation. Implements the transition process for production rules.

## [Lab 2: Determinism in FA, NDFA to DFA, Chomsky Hierarchy](lab2/)
**Description:** Classifying grammars according to the Chomsky hierarchy, checking for determinism in finite automata, and converting a Non-Deterministic Finite Automaton (NFA) to a Deterministic Finite Automaton (DFA) using the subset construction algorithm.
<br>
<img src="lab2/lab2_output.png">

---

## [Lab 3: Lexer & Scanner](lab3/)
**Description:** Implementation of a lexical analyzer (lexer/scanner) to transform a sequence of characters into meaningful tokens (such as identifiers, keywords, operators, and numerical values) from a mathematical expression or language schema.
